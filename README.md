# clean-up-sikkim

A community-driven web platform that empowers Indians to tackle local waste problems through crowdsourced action. Users can pin trash dump locations on an interactive map, making pollution hotspots visible to the community. Anyone can organize cleanup events at these tagged locations, and participants earn points for joining cleanups. These points accumulate into a rewards system that recognizes and incentivizes environmental action, turning waste management into a collaborative, gamified experience that motivates sustained community participation in keeping India clean.

This project is built using the AWS Serverless Application Model (AWS SAM) to leverage scalable, cost-effective cloud resources. The application uses AWS Lambda, API Gateway, DynamoDB, and other AWS services, all defined in the `template.yaml` file. See below for setup and deployment instructions.
